# restcountries_frontend

## Installation
 * Ensure you have node 12.x.x installed.

 * Clone the repo

 * cd into the project
 
 * Install node modules with the command
 
   * yarn install
   
 * Start the App
 
   * yarn start
   
 * Run test

   * yarn test
